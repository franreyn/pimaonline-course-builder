# PimaOnline Coure Builder

## Version

0.1.0

## What's included

### Dependencies

- Node.js
  - pnpm
  - local-server

### Terminal Command

`pnpm run dev` Runs local server at port `3000`  

### File Structure

`index.html`: Main file. App loads from this file.  

`config-lock.js`: Main configuation file, all module files are imported here.
